El desarrollo del proyecto se organiza en varias ramas para facilitar la colaboración:

main: Esta es la rama principal que contiene la versión unificada del proyecto. Aquí se integran las contribuciones de Pablo y Jennifer, cuando ambas partes están listas.

pablo12344-feature: Esta rama contiene las partes del proyecto desarrolladas por Pablo. Aquí sube los cambios y mejoras correspondientes a su trabajo.

jenn487-feature: Esta rama contiene las partes del proyecto desarrolladas por Jennifer. Aquí sube los cambios y mejoras correspondientes a su trabajo.
